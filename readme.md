## Toy_SavedForms

### 스택
Android, Kotlin

### 발견한 불편함
회원 가입할 때, 배송지 입력할 때 등 열심히 입력한 정보 중 일부가 날아간 적이 있었다. 
그래서 이번에 한 번 SavedInstanceState를 배운 김에 날아가지 않게 하는 방법을 사용해 보았다.
